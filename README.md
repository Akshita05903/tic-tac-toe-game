# Tic Tac Toe Game 🎮

A simple and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**.

## Features

* Two-player gameplay (X and O)
* Winner detection
* Draw detection
* Reset Game button
* New Game functionality
* Responsive and clean UI

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```
tic-tac-toe-game/
│── index.html
│── style.css
│── first.js
└── README.md
```

## How to Run

1. Clone the repository:

   ```
   git clone https://github.com/Akshita05903/tic-tac-toe-game.git
   ```

2. Open the project folder.

3. Run `index.html` in your browser.
